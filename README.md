# myBlog

## 简介

这是我的个人博客项目，使用Django框架搭建前后端，数据库使用的是MySQL。

由于开发尚未完成，暂未上线。

### 技术选型

**前端**：Django + Vue.js + Element UI

**后端**：Django + PyMysql + MySQL

### 创作背景

- 学习Django框架，个人兴趣
- 感觉程序员都人手一个个人博客，想做一个展示自己的网站
- 想进一步了解和学习一下后端

## 项目主要功能点

- [x] 主页
- [x] 新闻热搜
- [ ] 留言板
- [ ] 关于网站
- [ ] 外站导航
- [x] 搜索文章
- [x] 注册登录
- [x] 文章详情页
- [x] 文章发布与编辑
- [x] 评论点赞收藏
- [x] 后台管理
